Commands, Options, and Arguments

$ pass salt
$ pass --carefully salt
$ pass salt me

When using a shell like Bash, you enter commands that tell the computer what to do, using options that tell the computer how to do it, and arguments that describe what to do it to.

For example: "pass the salt" *CLICK* might be written as "pass salt"

If I were asking my three-year-old to hand me the salt shaker, I might add the option "carefully" to the command. *CLICK*

And if I were addressing a very literal-minded person I could add a second argument, just to be explicit: "pass the salt to me" *CLICK*
